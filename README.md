# Firdevs ❤️

Bu proje, Firdevs için hazırlanmış romantik bir özür ve sevgi web sitesidir.

## İçerik
- Arka plan fotoğrafı
- Müzik eşliğinde açılan sayfa
- Daktilo yazı efekti
- Kaçan "Hayır" butonu
- Kalp animasyonları
- Özel imza

## Dosyalar
- index.html → Ana web sayfası
- foto.jpg → Arka plan görseli
- muzik.mp3 → Arka plan müziği

## Kullanım
Bu site GitHub Pages üzerinden yayınlanmaktadır.

Sayfayı açtıktan sonra:
- Ekrana dokunarak müziği başlatabilirsiniz
- "Affettim" butonuna basıldığında animasyonlu kapanış görünür
